New Features

You might ask yourself what did the inspectIT guys work on since the last release. This release was once again quite a big one. We fixed over one hundred bugs and implemented as much as 105 improvements and new features. Please check the Release Notes 1.5 for in-depth information.

Improved monitoring of Web-application

inspectIT 1.4 provided basic metrics for Web-based performance diagnosis. Version 1.5 enhances the vision and allows the Web-page monitoring using the new "Display in Chart" feature. In addition, URIs can now be grouped into business terms using regular expression rules on the URI.

Installers for all platforms

inspectIT 1.5 ships with installers for Windows and Linux platforms. The installers ease up the installation of all inspectIT components, even on the terminal-only systems.

Storage 2.0

The storage feature has been improved and extended. Many users will be happy to know that storage functionalities can now be invoked by the RESTful service, making it easier to seamlessly integrate performance testing in continuous integration process. Besides this, there are many new cool features that you will like.

New UI-based CMR management

Complete management of the Central Management Repository (CMR) is now possible from the inspectIT UI. Changing the CMR settings, restarting or shutting-down the CMR, deleting the database data and much more is now available in "one-click".
